# Project Overview

## Product

**Portfolio** — Personal portfolio website for Richard O, a software engineer.

## Purpose

Showcase skills, featured projects, and contact information for professional networking and job opportunities.

## Site Sections

1. **Navigation** — Fixed header with smooth-scroll links to each section
2. **Hero** — Landing section with name, title, tagline, and CTAs
3. **About** — Brief bio and experience summary
4. **Skills** — Technology proficiencies grouped by category (Frontend, Backend, Tools)
5. **Projects** — Featured project cards with descriptions, tech tags, and links
6. **Contact** — Contact info (email, LinkedIn, GitHub) and a contact form

## Target Audience

- Recruiters and hiring managers
- Potential collaborators
- Fellow developers
